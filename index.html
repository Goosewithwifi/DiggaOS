Can I download it yet?

Hey there! I'm Goosewithwifi. I am building DiggaOS as a solo hobby project because I was bored and wanted to see how far I could push a custom Alpine base with the Cosmic Desktop environment.

Because it's just me working on this in my free time, it isn't quite ready for a public release just yet. I want to make sure it's completely stable and safe before publishing any images.

Early Alpha ISO & Desktop Builds

I'm currently custom compiling the Rust-based Cosmic applets to run seamlessly on top of Alpine's lightweight Musl-libc core. When it's ready, you'll be able to grab a live-bootable ISO file here to test in QEMU, VirtualBox, or on actual hardware.

Patience is a virtue!

I'm actively tying Alpine's musl layer with Cosmic. I will unlock downloads the instant I have a stable live image that boots safely in a VM. Hang tight!

No corporate backing, no telemetry, no tracking. Just code, coffee, and open-source curiosity.
